
Для сборки должны быть установлены:

face.vcxproj - проект без использования CUDA
face.cuda9.vcxproj - проект для CUDA 9.0
face.cuda10.vcxproj - проект для CUDA 10.0

Если используется CUDA и CUDNN, в окружении должна быть задана переменная CUDA_PATH

dlib http://dlib.net - должна находится в папке C:\dlib, но собирать нет необходимости, т.к. собранные файлы dlib.lib включены в этот проект.

Для OpenCV 4.1.0 в окружении должна быть задана переменная OPENCV_DIR

Параметры командной строки:
person <имя персоны> - всем обнаруженным персонам в изображениях назначается указанное имя (в изображениях должна присутствовать только единственная персона)
updаte - сохранение изменений (в файле persons.csv), если не указан, то никакие обновления по персонам на сохранятся (в изображениях могут присутствовать разные персоны)
<файл изображения 1> <файл изображения 2> ... - файлы изображений

В результате - создаётся или обновляется файл persons.csv (в каталоге face.exe).
Для каждого изображения рядом создаются файлы с отмеченными и подписанными лицами и текстового отчёта.

Примеры:

Для персон назначаются номера-идентификаторы и сохраняются:
face.exe update faces1.jpg faces2.jpg

Персоне в файле назначается имя:
face.exe update person1_faces.jpg person Name1
face.exe update person2_faces.jpg person "Фамилия Имя Отчество"

Теперь можно распознать и посмотреть изображения с разными персонами:
face.exe pic1.jpg pic2.png

В принципе можно номерам назначить имена вручную отредактировав файл persons.csv (имя указывается в последнем столбце).
Если в имене имеются пробелы то имя нужно взять в кавычки (кириллические символы при подписи на изображениях заменяются символом '?').
